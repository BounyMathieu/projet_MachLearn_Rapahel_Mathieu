## Accès aux données

Les données CGMacros ne sont pas disponibles dans ce dépôt.

**Accès restreint - certification obligatoire avant téléchargement.**

### Etape 1 - Créer un compte PhysioNet

1. Aller sur [https://physionet.org](https://physionet.org)
2. Cliquer **Sign Up**
3. Remplir le formulaire

---

### Etape 2 - Obtenir la certification CITI Program

PhysioNet exige une formation à l'éthique de la recherche sur sujets humains.

1. Aller sur [https://about.citiprogram.org](https://about.citiprogram.org)
2. Créer un compte CITI - **utiliser la même adresse mail que pour PhysioNet**
3. Sélectionner l'affiliation : choisir **"Independent Learner"** si vous n'avez pas d'université partenaire
4. Choisir le cours : **"Data or Specimens Only Research"**
5. Compléter tous les modules et obtenir le certificat
6. Télécharger le certificat PDF

---

### Etape 3 - Lier la certification à PhysioNet

1. Se connecter sur PhysioNet
2. Aller dans **Settings → CITI Program Certification**
3. Uploader le certificat PDF obtenu à l'étape 2
4. Attendre la validation (quelques heures à quelques jours)

---

### Etape 4 - Demander l'accès à CGMacros

1. Aller sur la page du dataset :
   [https://physionet.org/content/cgmacros/1.0.0/](https://physionet.org/content/cgmacros/1.0.0/)
2. Cliquer **"Request Access"**
3. Lire et signer le **Data Use Agreement (DUA)**
4. Accès accordé - téléchargement disponible
